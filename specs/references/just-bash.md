# just-bash reference

Source: https://justbash.dev/ (accessed 2026-03-03)

Summary:

- just-bash is a simulated Bash environment designed for use in browser/JS runtimes.
- It uses an in-memory virtual filesystem (InMemoryFs is the default).
- The docs recommend `bash-tool` as a ready-to-use AI SDK tool built on just-bash.

Additional source: https://github.com/vercel-labs/just-bash
