## Client-side

| Framework          | Support |
| ------------------ | ------- |
| React              | ✅      |
| Vue                | ✅      |
| Svelte             | ✅      |
| SolidJS            | ✅      |
| Vanilla JavaScript | ✅      |

<Callout type="info" title="Something missing?">
  If your favorite framework is missing, let us know by opening an [issue on
  Github!](https://github.com/rejot-dev/fragno/issues/)
</Callout>

## Server-side

| Framework               | Support     |
| ----------------------- | ----------- |
| Node.js / Express       | ✅          |
| React Router v7 / Remix | ✅          |
| Astro                   | ✅          |
| Next.js                 | ✅          |
| Nuxt                    | ✅          |
| SvelteKit               | ✅          |
| SolidStart              | ✅          |
| Tanstack Start          | ⚠️ Untested |

⚠️ Untested: should work using web standards, but not documented nor tested.

<Callout type="info" title="Web Standards">
  Fragno uses the web standard `Request` and `Response` objects, so frameworks _not_ listed here
  could be easily integrated regardless.
</Callout>
