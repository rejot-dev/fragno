`@fragno-dev/pi-fragment` runs Pi agents through durable workflow-backed sessions. It provides a
session API, agent/tool registration, event streaming, JSONL export, and custom workflow support for
multi-step agent orchestration.

## Start here

- [Custom workflows](./custom-workflows): define durable agent workflows with `definePiWorkflow`,
  `ctx.agent(...)`, tool-result branching, and parallel reviewer patterns.
