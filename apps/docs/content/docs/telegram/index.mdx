## Overview

The Telegram fragment gives you a production-ready bot backend: webhook intake, typed command
handlers, chat and member tracking, and a client SDK to query chats or send messages.

**Use it when you need:**

- A reliable webhook ingestion pipeline with deduped updates
- A command registry that lives next to your product logic
- A database-backed history of chats, members, and messages
- Hooks to trigger workflows when messages arrive
